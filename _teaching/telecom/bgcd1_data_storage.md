## Course overview
Introduction to common professional data storage formats including CSV, JSON, GeoJSON, XML, and Excel. The course also presents the fundamental concepts of relational and non-relational databases and associated storage challenges.

## Course objectives
 - Provide an overview of storage technologies.
 - Learn about the different file formats used in the professional world for storing data.
 - Know how to define a database.
 - Understand the implications of storing data in a database.
